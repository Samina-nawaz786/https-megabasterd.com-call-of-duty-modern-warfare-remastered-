Call of Duty Modern Warfare Remastered is a revamped version of the classic Call of Duty 4 Modern Warfare, developed by Raven Software and published by Activision. Released in 2016, this remastered edition stays true to the original game while introducing enhanced visuals, improved textures, and updated audio.
